## PERCEPTRON AND ADALINE ALGORITHM IMPLEMENTATION

### Running the program

install python (_3.6+ recommended_).  
install pipenv globally.  
navigate to project root directory.  
run: **pipenv shell && pipenv install**.  
run: **python main.py**.
